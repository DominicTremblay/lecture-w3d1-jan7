2em Headings h1..h6, paragraphs, and ul lists are block-level elements. tags such as strong, Emphasis or span are inline elements. Block-level elements have a lightblue background while inline elements are in pink

Some key characteristics of block elements

  • Take the full width of the horizontal space available defined by its parent container
  • You can set box properties to change their appearance
